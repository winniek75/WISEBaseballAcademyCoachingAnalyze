# WISE Baseball Coaching Analyzer

WISE Baseball Academy 内部コーチング動画分析ツール

## 機能

- スロー再生（1/10 〜 等速）
- コマ送り（±1フレーム）
- 描画ツール：矢印・直線・楕円・四角・フリーハンド・角度計測
- カラー7色 / 線の太さ3段階
- アンドゥ / リドゥ
- フレーム＋描画をPNG書き出し

## 使い方

```bash
npm install
npm run dev
```

## デプロイ（Vercel）

GitHubにプッシュ後、Vercelでリポジトリを連携するだけで自動デプロイされます。

## Tech Stack

- React 18
- Vite 5
- HTML5 Canvas API
- HTML5 Video API
